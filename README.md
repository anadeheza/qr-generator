# QR Code Generator
This is just s simple page to generate a QR code on anything you want
<br>
Made with HTML, CSS and Python! (and a little bit of javascript for the downloading part). Mainly to start learning how to use Python 

## How to use
The site is pretty straighforward, just add the URL or thext you want to use and click "Generate", the QR code will appear right under that, along with clear and download buttons

Download: click there if you want to save your QR in your device
<br>
Clear: To reset the generator just click here and the code will dissappear
